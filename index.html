Documentação da API

Consulta em Lote (POST)

Permite consultar vários números simultaneamente.

https://sms.witi.me/izdata/hlr.aspx?chave=SUA_CHAVE_API

Estrutura de Entrada (Body)

Uma lista (JSON Array) de números de telefone em formato de string.

["54992389592", "11987654321", ...]

Estrutura da Resposta (JSON)

A resposta contém uma lista de telefones e o resultado da consulta.

Campos do Objeto "Telefone"

Propriedade Descrição
Valido Indica se o telefone é válido (true/false).
Operadora Nome da operadora.
WhatsApp Indica se há um WhatsApp registrado (true/false).
DataConsulta A data da última atualização da informação no banco de dados da API (cache), e não a data em que você fez a sua consulta.
WhatsAppDataStatus A data do status retornado no campo `WhatsApp`. O valor "0001-01-01" é um valor nulo.
